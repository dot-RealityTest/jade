Jade

Native macOS terminal workspace — project tabs, split panes, Git worktrees, command palette, local Ollama AI, and Obsidian capture. Built with SwiftUI and libghostty.

Download GitHub Documentation

Project-first terminal

Organize shells by repo with persistent tabs, splits, and worktrees — not endless loose windows.

Command palette & Git

⌘K fuzzy actions, file search, built-in VCS, diff viewer, and GitHub PR workflows via gh.

Local AI & capture

Ollama assistant, Rich Input, snippets, on-device voice, and Obsidian vault export through MCP.

Requirements & install

macOS 14+. Download a release or clone the repo, run scripts/setup.sh, then ./scripts/run-jade.sh. MIT licensed — feedback and contributions welcome.
